name: milestones
describe: work with the user to identify the delivery milestones for a roadmap
---

Work with the user to extract the milestones for a given roadmap `$ARGUMENTS`.

Each milestone aims to deliver value to the product and incrementally introduce features. Each milestone should describe the capability that it achieves after completion.

Milestones are separate from the roadmap — they are the incremental steps within each roadmap phase. Once agreed, write them to `plans/milestones.md` as a standalone document that references the roadmap.

Consider the information in the [plans/](plans) directory.
